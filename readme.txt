A note-taking and management plugin for organizing notes by workspace, with REST API support for easy access.

== Description ==

Notepress is a WordPress plugin designed for organizing and managing notes using custom post types and taxonomies. It includes a custom REST API endpoint for creating, retrieving, and managing notes, which can be organized by workspaces.

**Features:**
* Create, update, delete, and search notes.
* Organize notes by workspaces for easy filtering.
* REST API endpoints for managing notes and workspaces programmatically.

== Installation ==

1. Upload the `notepress` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the available REST API endpoints to interact with notes.

Notepress is a plugin for managing notes in WordPress. It allows users to categorize notes by workspaces and provides REST API endpoints for managing these notes.
